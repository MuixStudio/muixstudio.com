## Quick Start

### Manual installation

1. Clone the repository.

```sh
git clone https://github.com/MuixStudio/muixstudio.com.git
cd muixstudio.com
```

2. Install the project's dependencies.

```sh
pnpm install
```

3. Run on dev

```sh
pnpm run dev
```
